# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.1.x   | ✓         |
| 1.0.x   | security fixes only |
| < 1.0   | ✗         |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report them by email to **security@nimblyweb.com**. Include a description of the issue, steps to reproduce, and the potential impact.

You will receive an acknowledgement within 48 hours. We will work with you to understand and fix the issue before any public disclosure.
